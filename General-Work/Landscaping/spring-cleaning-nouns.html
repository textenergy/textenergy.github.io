Spring Cleaning: Nouns / Limpieza de primavera: Sustantivos

Note / Nota: You will hear an English word. Find and tap the matching Spanish card. When correct, the card flips to show both words. We recommend Chrome or Edge browser for good pronunciation.

Escucharás una palabra en inglés. Encuentra y toca la tarjeta en español que corresponde. Cuando sea correcta, la tarjeta se voltea para mostrar ambas palabras. Recomendamos los navegadores Chrome o Edge para una buena pronunciación.

▸ Device setup & progress saving / Configuración del dispositivo

Android: Settings → Language & Input → Text-to-speech → Install English voice
Android: Ajustes → Idioma y entrada → Texto a voz → Instalar voz en inglés
iOS: Settings → Accessibility → Spoken Content → Voices → English
iOS: Ajustes → Accesibilidad → Contenido hablado → Voces → Inglés

Progress is saved on this device. Private/incognito browsing does not save progress.
El progreso se guarda en este dispositivo. La navegación privada/incógnito no guarda el progreso.

✓ Progress saved! Click "Reset" to start again with 0 score.
¡Progreso guardado! Haga clic en "Reiniciar" para comenzar de nuevo con puntuación 0.
