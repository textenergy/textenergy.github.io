# textenergy.github.io

*Explorations in bilingual English/Spanish learning materials and related UI*

I tutor Spanish speakers in English for general conversation, specific work-related vocabulary and US citizenship preparation. My resources are intended to be functional HTML sets with supporting CSS and minimal JS to provide interactivity. The current site is mostly for Citizenship prearation, but more general content will be added after I finish working on the citizenship content.
Anyone will be welcome to download the HTML, CSS and JS files to use for developing their own materials, with attribution to this source repository.
As I develop the site, I may revise files, so please check the commits and grab the latest files to download if you will work with them offline.
